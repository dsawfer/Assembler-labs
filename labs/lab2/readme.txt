ab2.asm - main program (laboratory).
deleter.asm - program for deleting files.

Pressing the f1 button brings up a menu with a choice of file types. The user uses the up and down arrows to select the type of file. The user uses "Enter" to confirm the selection.

The window disappears only after pressing Enter. If the user does not want to prohibit the deletion of any file types, select the "none" item and press Enter.

To delete a file, you need to type the command "deleter File_Name.File_Type".
